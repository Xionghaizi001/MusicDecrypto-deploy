# MusicDecrypto Deploy

This repository is a monorepo for MusicDecrypto deployment code.

Project details live in the packages:

- `packages/frontend` - frontend application
- `packages/backend` - backend service and deployment scripts

Please read the README files in those package directories for setup, build, and deployment notes.
